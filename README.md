# Lampa-Plex

Плагин для [Lampa](https://github.com/yumata/lampa-source), показывающий медиатеку личного Plex-сервера (фильмы и сериалы) и запускающий воспроизведение через Lampa.Player — с прозрачной передачей во внешний плеер, если он выбран в настройках Lampa.

## Возможности

- Раздел **«Plex»** в левом меню (значок — тот же play-в-квадрате, что и у кнопки «Смотреть из Plex»): сразу открывается объединённая сетка карточек из **всех** выбранных медиатек (фильмы и сериалы вперемешку), с постерами из Plex. Сверху — строка фильтров (по образцу «Хочу посмотреть» из LampaTrakt, в цветах Plex): **«Медиатека»** (по умолчанию «Все» — сужает список до одной конкретной медиатеки или запускает поиск, который ищет сразу по всем выбранным медиатекам, если медиатека не сужена), **«Год»** (конкретный год или диапазон), **«Жанр»**, **«Страна»** (списки — реальные значения из вашей медиатеки) и **«Сортировка»** (по дате добавления/названию/дате выхода/длительности/рейтингу/моей оценке/дате просмотра/случайно — повторное нажатие того же пункта переключает направление).
- **Гибридные карточки**: если у элемента Plex есть сопоставление с TMDB, открывается родная карточка Lampa (описание, актёры, похожее) с добавленной кнопкой **«Смотреть из Plex»** — обычный источник в общем списке «Смотреть» (рядом с «Торренты»/«Онлайн»), со значком Plex. Пока фильм/сериал доступен в Plex, эта кнопка автоматически закрепляется отдельной кнопкой на карточке (штатным механизмом Lampa — как если бы вы сами долго нажали на неё в списке источников); на карточках без Plex закрепление возвращается к тому, что было до этого (другой источник или ничего). Появляется на **любой** карточке фильма/сериала, доступного в вашей медиатеке — независимо от того, как вы на неё попали (поиск, рекомендации, раздел «Plex» и т.д.), как только собран кэш медиатеки (см. ниже). Если сопоставления с TMDB нет — показывается минимальная карточка на основе данных Plex (постер, описание, год, кнопка воспроизведения; для сериалов — выбор сезона и серии).
- **Direct Play**: прямая ссылка на файл с сервера передаётся в `Lampa.Player.play`, дальше Lampa сама решает — проигрывать внутри приложения или передать во внешний плеер (если так настроено в самой Lampa).
- **Синхронизация прогресса с Plex**: при закрытии плеера позиция и статус просмотра отправляются на сервер (`/:/timeline`, `/:/scrobble`); при повторном открытии Lampa подхватывает сохранённую позицию (`viewOffset`) для продолжения просмотра.
- **Раздел настроек Plex**: адрес сервера и токен (вручную или через вход по коду на plex.tv с автоматическим определением сервера), выбор отображаемых медиатек (по умолчанию выбраны все — снимайте галки, чтобы скрыть ненужные), сортировка по умолчанию для раздела «Plex» (повторный выбор того же пункта переключает направление, как и кнопка «Сортировка» внутри самого раздела), включение/выключение синхронизации просмотра.
- **Кэш медиатеки**: чтобы кнопка «Смотреть из Plex» появлялась на карточке сразу, без предварительного захода в раздел «Plex», плагин хранит локальную карту «TMDB ID → элемент Plex» (`Настройки → Plex → Обновить кэш медиатеки`, там же видно, когда обновлялся и сколько сопоставлено). Тихо (без уведомлений) пересобирается при каждом запуске Lampa, а также сразу после изменения списка медиатек или вручную по кнопке.
- **Статусы Trakt.TV** (если установлен и авторизован [LampaTrakt](https://github.com/KZuev/LampaTrakt)): пункт «Статусы Trakt.TV» в `Настройки → Plex → Воспроизведение` — при включении «просмотрено» в интерфейсе плагина (карточка фильма/сериала, список серий) берётся из активного аккаунта Trakt вместо Plex; если запрос к Trakt по какой-то причине не удался, статус автоматически берётся из Plex, как и раньше. Trakt отмечает серии как просмотрено/не просмотрено (без «процента досмотра» в середине серии — это концепт Plex), поэтому в списке серий вместо процента показывается метка «Просмотрено». Рядом — пункт **«Проверить подключение к Trakt»**, показывающий, найдены ли учётные данные и успешен ли последний запрос (с числом фильмов/сериалов или текстом ошибки) — полезно, если статусы выглядят пустыми. Появляется отдельная кнопка **«Синхронизировать статусы в Plex»** — по подтверждению разово отмечает в Plex как просмотренные фильмы и полностью просмотренные сериалы из истории Trakt (частично просмотренные сериалы не трогает — у Trakt нет стабильной поэпизодной разбивки, чтобы делать это надёжно). Действие ручное, необратимо без отдельной отметки «не просмотрено» в самом Plex.
- **Отправлять просмотр в Trakt** (тумблер в `Настройки → Plex → Воспроизведение`, если установлен LampaTrakt): после просмотра через Plex отмечает фильм/серию просмотренными в активном аккаунте Trakt. Порог, при котором просмотр считается досмотренным, берётся из настройки самого LampaTrakt («Порог просмотра»), чтобы Plex и Trakt отмечали по одному правилу; ниже порога — сохраняет позицию в Trakt. Работает для tmdb-сопоставленного контента. Если отправка не удалась (протух токен Trakt, нет сети), отметка сохраняется локально и **автоматически до-отправляется при следующем запуске Lampa** — пересматривать серию не нужно. Токен Trakt обновляет сам LampaTrakt.

## Установка

1. В Lampa откройте **Настройки → Плагины → Добавить плагин вручную**.
2. Вставьте ссылку на файл `lampa-plex.js`:
   - `https://kzuev.github.io/Lampa-Plex/lampa-plex.js` (публикуется автоматически через GitHub Actions при каждом пуше в `main`), либо
   - raw-ссылка на файл в этом репозитории.
3. Перезагрузите Lampa.
4. Откройте **Настройки → Plex** и укажите адрес сервера + токен, либо нажмите **«Войти через Plex.tv»**.
5. В настройках выберите, какие медиатеки показывать («Выбрать медиатеки»).
6. В левом меню появится пункт **«Plex»**.

### Получение токена вручную

Если не хотите входить через plex.tv — найдите свой `X-Plex-Token` по инструкции [Finding an authentication token / X-Plex-Token](https://support.plex.tv/articles/204059436-finding-an-authentication-token-x-plex-token/) и укажите его вместе с адресом сервера (например `http://192.168.1.50:32400`) в настройках плагина.

## Известные ограничения (MVP)

- **Только Direct Play.** Если кодек файла (например HEVC/DTS) не поддерживается плеером Lampa или внешним плеером на устройстве — воспроизведение не начнётся. Серверное транскодирование Plex в этой версии не реализовано.
- **Субтитры** — поддерживаются только внешние (sidecar) дорожки, которые Plex может отдать напрямую по ссылке. Встроенные в контейнер субтитры (например PGS-картинки) не извлекаются без транскодирования.
- **Синхронизация прогресса при внешнем плеере** зависит от того, возвращает ли конкретный внешний плеер позицию обратно в Lampa (`Lampa.Timeline`) — так же, как и для остальных источников в Lampa.
- Домашние пользователи (Plex Home) и несколько серверов на один аккаунт: при входе через plex.tv можно выбрать один сервер из списка; переключение между серверами — через повторный вход или ручную смену адреса в настройках.
