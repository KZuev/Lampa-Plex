# Lampa-Plex

Плагин для [Lampa](https://github.com/yumata/lampa-source), показывающий медиатеку личного Plex-сервера (фильмы и сериалы) и запускающий воспроизведение через Lampa.Player — с прозрачной передачей во внешний плеер, если он выбран в настройках Lampa.

## Возможности

- Раздел **«Медиатека Plex»** в левом меню: список ваших библиотек (фильмы/сериалы) → сетка карточек с постерами из Plex, поиск по названию.
- **Гибридные карточки**: если у элемента Plex есть сопоставление с TMDB, открывается родная карточка Lampa (описание, актёры, похожее) с добавленной кнопкой **«Смотреть в Plex»** — она выглядит и ведёт себя как обычный источник в меню «Смотреть» (рядом с «Торренты»/«Онлайн»), со значком Plex. Кнопка появляется на **любой** карточке фильма/сериала, доступного в вашей медиатеке — независимо от того, как вы на неё попали (поиск, рекомендации, «Медиатека Plex» и т.д.), как только собран кэш медиатеки (см. ниже). Если сопоставления с TMDB нет — показывается минимальная карточка на основе данных Plex (постер, описание, год, кнопка воспроизведения; для сериалов — выбор сезона и серии).
- **Direct Play**: прямая ссылка на файл с сервера передаётся в `Lampa.Player.play`, дальше Lampa сама решает — проигрывать внутри приложения или передать во внешний плеер (если так настроено в самой Lampa).
- **Синхронизация прогресса с Plex**: при закрытии плеера позиция и статус просмотра отправляются на сервер (`/:/timeline`, `/:/scrobble`); при повторном открытии Lampa подхватывает сохранённую позицию (`viewOffset`) для продолжения просмотра.
- **Раздел настроек Plex**: адрес сервера и токен (вручную или через вход по коду на plex.tv с автоматическим определением сервера), выбор отображаемых библиотек, включение/выключение синхронизации просмотра.
- **Кэш медиатеки**: чтобы кнопка «Смотреть в Plex» появлялась на карточке сразу, без предварительного захода в «Медиатеку Plex», плагин хранит локальную карту «TMDB ID → элемент Plex» (`Настройки → Plex → Обновить кэш медиатеки`). Собирается автоматически при первом запуске и обновляется раз в 12 часов, либо вручную по кнопке или сразу после изменения списка библиотек.

## Установка

1. В Lampa откройте **Настройки → Плагины → Добавить плагин вручную**.
2. Вставьте ссылку на файл `lampa-plex.js`:
   - `https://kzuev.github.io/Lampa-Plex/lampa-plex.js` (публикуется автоматически через GitHub Actions при каждом пуше в `main`), либо
   - raw-ссылка на файл в этом репозитории.
3. Перезагрузите Lampa.
4. Откройте **Настройки → Plex** и укажите адрес сервера + токен, либо нажмите **«Войти через Plex.tv»**.
5. В настройках выберите, какие библиотеки показывать («Выбрать библиотеки»).
6. В левом меню появится пункт **«Медиатека Plex»**.

### Получение токена вручную

Если не хотите входить через plex.tv — найдите свой `X-Plex-Token` по инструкции [Finding an authentication token / X-Plex-Token](https://support.plex.tv/articles/204059436-finding-an-authentication-token-x-plex-token/) и укажите его вместе с адресом сервера (например `http://192.168.1.50:32400`) в настройках плагина.

## Известные ограничения (MVP)

- **Только Direct Play.** Если кодек файла (например HEVC/DTS) не поддерживается плеером Lampa или внешним плеером на устройстве — воспроизведение не начнётся. Серверное транскодирование Plex в этой версии не реализовано.
- **Субтитры** — поддерживаются только внешние (sidecar) дорожки, которые Plex может отдать напрямую по ссылке. Встроенные в контейнер субтитры (например PGS-картинки) не извлекаются без транскодирования.
- **Синхронизация прогресса при внешнем плеере** зависит от того, возвращает ли конкретный внешний плеер позицию обратно в Lampa (`Lampa.Timeline`) — так же, как и для остальных источников в Lampa.
- Домашние пользователи (Plex Home) и несколько серверов на один аккаунт: при входе через plex.tv можно выбрать один сервер из списка; переключение между серверами — через повторный вход или ручную смену адреса в настройках.
